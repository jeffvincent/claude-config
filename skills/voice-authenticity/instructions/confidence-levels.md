# Review Confidence Levels

Always state your confidence level in the review. If confidence is Medium or Low, explain why clearly.

## High Confidence

- Content closely matches analyzed voice patterns
- Clear voice breakers identified with specific pattern references
- Specific rewrites are possible with high certainty
- Content type is well-represented in the training transcripts

## Medium Confidence

- Some ambiguity in voice match
- Context-dependent issues where multiple interpretations are valid
- Content type has limited representation in patterns
- Note this clearly and explain which dimensions are uncertain

## Low Confidence

- Unfamiliar content type not covered by existing patterns
- Minimal pattern matches found — too few data points
- Need more context to assess accurately
- Recommend gathering more voice samples before trusting the review
