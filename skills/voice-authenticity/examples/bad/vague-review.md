# Anti-Pattern: Vague Review

This shows what a BAD voice authenticity review looks like. Avoid this.

---

## What's wrong with this review

1. **No specific pattern references** — just vibes
2. **No concrete rewrites** — tells the user something is wrong but not how to fix it
3. **No location markers** — user has to hunt for the problems
4. **No metrics** — everything is qualitative handwaving
5. **No authentic moments** — all criticism, no encouragement

---

## Bad Review Example

> ### Overall Assessment
> **Rating:** Needs Work
>
> ### Issues
> 1. Some of the language feels a bit corporate
> 2. A few sentences are too long
> 3. The tone shifts in the middle
> 4. Consider making it more conversational
> 5. Some words could be more natural
>
> ### Recommendations
> - Try to sound more like yourself
> - Shorten some sentences
> - Use simpler words

---

## Why this fails

- **"Feels a bit corporate"** — Which sentences? Which words? What pattern does it violate?
- **"A few sentences are too long"** — Which ones? What's the current length vs. target?
- **"The tone shifts"** — From what to what? In which paragraph?
- **"More conversational"** — What does that mean specifically for this user's voice?
- **"Simpler words"** — Which words? What should they become?

Every observation should be: **[Location] + [Current text] + [Specific pattern violated] + [Concrete rewrite]**
